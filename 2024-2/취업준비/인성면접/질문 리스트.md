######  1분 자기소개
    
    
    경험, 기술, 업무 방식, 목표와 비전을 구체적으로 언급하고, 차별점을 보여주자
    
    경험 : 경력을 설명하되, 질적으로 임팩트 있는 부분
    
    기술 : 특정 기술을 어떻게 활용해왔는지
    
    업무 방식 : 협업에 강점이 있거나, 문제 해결에 집중하는 스타일 등을 언급
    
    목표와 비전 : 어떤 개발자, 사람이 되고 싶은지 짧고 명확하게

예시
    저는 5년 차 프론트엔드 개발자입니다.
    
    주로 React와 Vue.js를 사용해 UI를 구성하며, 사용자 경험을 개선하는 데 중점을 두고 있습다.
    
    이전 회사에서는 대형 쇼핑몰 사이트 리뉴얼 프로젝트에 참여해,
    
    페이지 로딩 시간을 30% 단축한 경험이 있습니다.
    
    팀원들과 코드 리뷰를 자주 진행하며 협업하는 걸 중요시하는데요,
    
    앞으로는 성능 최적화와 인터랙티브한 UI 디자인에 더 집중하고 싶습니다

저는 백엔드 개발자입니다
Spring 프레임워크를 사용해 서버 개발을 하며 확장성과 안정성을 중요하게 생각합니다.
제가 참여했던 프로젝트에서는 Spring Security를 사용해 유저 관련 기능을 개발한 경험이 있습니다. 저는 사고가 갇히는 걸 피하는 것을 중요시 해 팀원들과의 소통을 중요시합니다.
기아에서의 인턴 실무 경험을 통해 전문가로 거듭나고 싶습니다.

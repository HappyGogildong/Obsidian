블록체인 네트워크 구조
	분산원장 : 모든 참여자에게 복제되어 저장. 무결성, 투명성 보장
	블록과 체인 
	노드

참여자간의 상호 작용
	거래 생성 전파
	거래 검증 합의
	블록체인 데이터 동기화

참여자들의 역할
	사용자 : 거래 생성, 블록체인 서비스 이용
	노드 운영자 : 블록체인 네트워크 유지, 거래 검증
	개발자 : 블록체인 기반 앺, 서비스 개발
	채굴자 : 작업 증명 방식의 블록체인에서 거래 검증, 블록 생성

블록체인 네트워크의 특징
	탈중앙화
	투명성
	보안성
	불변성

네트워크 : 다양한 장치들이 서로 연결되어 데이터를 주고받는 통신 시스템
	단말, 전송매체(광섬유, wifi 등)로 이루어짐

네트워크 장비 : 네트워크 구성, 관리하는 장치
	라우터 스위치 허브 방화벽
	프로토콜 : 네트워크 소속 장치들이 통신하기 위한 규칙, 약속
		tcp/ip, http ..

네트워크 토폴로지(구조) 분류
	버스형 : 하나의 케이블에 여러 장치. 간단한 설치, 케이블 문제 -> 전체 마비
	스타형 : 중앙허브 or 스위치 중심 연결. 한 장치의 장애가 다른 장치에 영향 x
	링형 : 장치들이 고리 모양 연결. 한 방향ㅇ으로 순회하며 전송
	망형 : 모든 장치들 상호 연결. 안정성 높으나, 비용 많이 듦

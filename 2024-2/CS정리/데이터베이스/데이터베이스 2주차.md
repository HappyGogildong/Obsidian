데이터베이스 : 한 조직의 여러 응용 시스템들이 공용하기 위해 통합, 저장한 운영 데이터의 집합
	통합된 데이터 : 한 곳에 하나만 저장되게(중복 x)
	저장 데이터
	운영 데이터
	공용 데이터

데이터베이스의 특징
	온라인 접근성(Online Accessibilities) : 실시간 처리 응답
	계속 변화 : 삽입, 삭제, 갱신으로 상태 유지
	동시 공용 : 여러 사용자가 동시에 사용 v
	내용에 의한 참조

데이터베이스 관리 시스템(DBMS) : 응용 프로그램과 데이터의 중재 역할
	정의 기능 : 데이터베이스의 구조 정의.(Data Definition)
		데이터의 논리적 구조 명세
		물리적 저장 구조 명세
		물리/논리적 사상 명세
	조작 기능 : 검색, 갱신, 삽입, 삭제(Data Manipulation). 데이터는 read, write(insert, delete, update)
	제어 기능 : 데이터의 정확성, 안정성 유지 기능
		삽입, 삭제, 갱신 간의 무결성 유지
		정당한 사용자 검증
		동시 접근 시 정확성 유지(concurrency control)

데이터베이스 시스템 사용자
	최종 사용자(End User) : 
		일반 사용자(Casual User)
		데이터베이스 지식 가진 사용자
	데이터베이스 설계자 : 
		명세서 제작
		정보 내용 명세, 데이터베이스 구축
		개념적,논리적,물리적 설계
	응용 개발자
		데이터베이스 기능 이용해 프로그램 설계 개발
	데이터베이스 관리자
